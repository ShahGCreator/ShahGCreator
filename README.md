# 💫 About Me:
Hi, I'm a **Free Fire Craftland Creator**.  
I design and build creative maps in Craftland and also teach others how to create and publish their own maps.

Besides that, I am a **Programmer** who enjoys coding and learning new technologies.  
I also do **Data Mining**, where I explore and analyze game files and data to understand how systems work.

## My Skills
- Free Fire Craftland Map Creating
- Craftland Teaching
- Programming & Coding
- Data Mining
- Learning New Technologies

## My Goal
My goal is to **create, teach, and explore new things in gaming and programming**.
